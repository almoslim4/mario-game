# Pipebound Panic

An original browser platformer inspired by classic jump-and-stomp games. Open `index.html` in a browser and play.

## Controls

- Move: `A/D` or arrow keys
- Jump: `W`, `Up`, or `Space`
- Dash: `Shift` or `K` after collecting the dash pepper
- Pause: `P`
- Restart current world: `R`
- Use the side buttons for sound, fullscreen, Assist mode, and Zen mode

## Goal

Collect coins, sky gems, powerups, and hearts while crossing five handmade worlds. Chain stomps, bounce pads, dash rings, and risky aerial routes to raise Flow for bigger scoring. Magnet charms pull coins and gems toward you for a short burst, stars let you crash through danger, and checkpoint flags make harder routes worth trying.

## Prize Blocks

Jump into a `?` block from below to open it. If bricks or terrain sit directly under a `?` block, bump that support block from below and the prize above will pop open too.

## Play Options

Assist mode starts runs with more hearts, adds extra jump forgiveness, softens hit knockback, and saves you from losing a heart on falls. Zen mode keeps the same game but reduces particles, motion streaks, and camera shake for a calmer run.
